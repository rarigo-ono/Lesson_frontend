JSON

JSONは、データの保存や転送をするための形式

データがサーバーからWebページに送信されるときによく使用する。

JSONとは？

  • Java Script Object Notaiion の略
  • 軽量のデータ交換形式
  • 言語に依存しない
  • 「自己記述型」で理解しやすい

JSONの例

{
"employees":[
　{"firstName":"john","lastName":"Doe"}, {"firstName":"Anna","lastName":"Smith"}, {"firstName":"Peter","lastName":"Jones"}
]
}

JSON形式はJavaScriptオブジェクトとして評価される

JSON形式は、JavaScriptオブジェクトを作成するコードと構文的に同じである。

この類似性により、JavaScriptプログラムはJSONデータをネイティブJavaScriptオブジェクト に簡単に変換できる。

JSON構文ルール

  • 名前と値のペアのデータである。
  • データはカンマ , で区切られる
  • 中括弧はオブジェクトを保持する。
  • 角括弧は配列を保持する。

名前と値

名前と値のペアは、フィールド名(二重引用符で囲んだ)、 コロン、値で構成される。

"firstName":"John"

オブジェクト

オブジェクトは中括弧内に記述する。

JavaScript同様、複数の名前と値を含めることができる。

{"firstName":"John", "lastName":"Doe"}

配列

配列は、角括弧内に記述する。

"employees":[
{"firstName":"John","lastName":"Doe"}, {"firstName":"Azuki","lastName":"Ono"}, {"firstName":"Keisuke","lastName":"Ono"}
]
